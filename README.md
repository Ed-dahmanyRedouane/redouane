# BookShop API 📚

A robust RESTful API for an online bookstore, developed as part of a DevOps and Spring Boot practical exam.

## 🚀 Features
* **Public Access:** Browse books and categories with pagination.
* **Secure Authentication:** JWT-based login and registration.
* **Shopping Cart:** Protected cart management for authenticated users.
* **Admin Dashboard:** Role-based access control (RBAC) for managing the book inventory.

## 🛠️ Tech Stack
* **Backend:** Java 21, Spring Boot 3.3.
* **Security:** Spring Security, JWT (jjwt).
* **Database:** MySQL 8, Spring Data JPA.
* **DevOps:** Docker, Docker Compose, GitHub Actions (CI/CD).
